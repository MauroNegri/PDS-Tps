## ----------------------------------------------------------------
## CASO 1: misma señal - PI = 50
## ----------------------------------------------------------------
## - Las dos señales se superponen exactamente (una tapa a la otra)
## - El producto punto a punto x1*x2 = sin^2(x) es SIEMPRE positivo
##   nunca baja de 0, oscila entre 0 y 1
## - Al sumar todos esos valores positivos se obtiene PI = N/2 = 50
## - Conclusion: señales identicas → PI maximo positivo
##
## ----------------------------------------------------------------
## CASO 2: distinta amplitud - PI = 150
## ----------------------------------------------------------------
## - Se ven claramente las dos senoidales con distinta amplitud
##   x3 (A=1) y x4 (A=3)
## - El producto x3*x4 = sin(x)*3*sin(x) = 3*sin^2(x)
##   tambien es SIEMPRE positivo, pero 3 veces mas grande que caso 1
## - PI = 150 = 3 * 50, escala linealmente con la amplitud
## - Conclusion: mayor amplitud → mayor PI, pero misma forma
##   por eso es importante NORMALIZAR para comparar correctamente
##
## ----------------------------------------------------------------
## CASO 3: fase opuesta (pi) - PI = -50
## ----------------------------------------------------------------
## - Se ven las dos señales perfectamente opuestas
##   cuando x5 sube, x6 baja exactamente igual
## - El producto x5*x6 = sin(x)*(-sin(x)) = -sin^2(x)
##   es SIEMPRE negativo, oscila entre -1 y 0
## - Al sumar todos esos valores negativos: PI = -N/2 = -50
## - Conclusion: señales opuestas → PI maximo negativo
##   es el caso de "menos parecido" posible
##
## ----------------------------------------------------------------
## CASO 4: fase pi/2 (ortogonales) - PI = 2.609e-15 ≈ 0
## ----------------------------------------------------------------
## - Se ven seno y coseno, desplazados pi/2 entre si
## - El producto x7*x8 = sin(x)*cos(x) = (1/2)*sin(2x)
##   ALTERNA entre positivo y negativo en partes IGUALES
##   por eso en la grafica se ve una senoidal centrada en 0
## - Al sumar: los positivos y negativos se cancelan → PI ≈ 0
## - El valor 2.609e-15 es error numerico de punto flotante, no es PI=0 exacto
## - Conclusion: desfase de pi/2 → señales ortogonales, sin parecido
##
## ----------------------------------------------------------------
## CASO 5: distinta frecuencia (ortogonales) - PI = 1.1428e-14 ≈ 0
## ----------------------------------------------------------------
## - Se ven dos senoidales de distinta frecuencia (5 Hz y 3 Hz)
## - El producto x9*x10 es mas complejo visualmente:
##   tiene zonas positivas y negativas que NO son iguales en tamaño
##   pero al integrar en un periodo completo se cancelan
## - PI ≈ 0, aunque la forma del producto es irregular
## - Conclusion: senoidales de distinta frecuencia son ORTOGONALES
##   este es el fundamento de la Transformada de Fourier:
##   cada frecuencia es "independiente" de las demas
##   el PI permite "detectar" cuanta energia hay en cada frecuencia

## El producto punto a punto (subplot inferior) es la clave visual:
## SIEMPRE POSITIVO  → PI grande positivo  → señales muy parecidas
## SIEMPRE NEGATIVO  → PI grande negativo  → señales opuestas
## ALTERNA ±         → PI ≈ 0             → señales ortogonales
## Y el valor del PI depende de:
## - la FASE:       0 → max, pi/2 → 0, pi → min
## - la AMPLITUD:   escala el PI proporcionalmente
## - la FRECUENCIA: distinta frecuencia → siempre ortogonales
